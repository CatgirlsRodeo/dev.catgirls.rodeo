{% include article_title.html color="white" id="rodeo_kit" title="Rodeo Kit" %} {% include article_begin.html size="xl" color=code.color %} {% capture myInclude %} Rodeo Kit is a collection of game development libraries that act as a hobby game engine when combined together. It is still in the early months of developments so there are no complete examples or games yet other then the basic development stress test. #### The main goals of this project are: - To learn more and research about the low level components of graphics and game development. - Having a Web Browser export in addition to a Desktop export(Windows and Linux) - Fast development velocity and iteration times when developing games. - Using C for performance critical sections and embedding mruby for iterating on game mechanics. - Documented C and Ruby API, as well as automated building and deployment of accessable documentation. {% endcapture %} {{ myInclude | markdownify }} {% include article_end.html %}
